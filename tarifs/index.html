Cours particuliers

Vous souhaitez vous perfectionner plus particulièrement dans une danse, réviser certaines figures déjà apprises, apprendre à danser la valse ou la salsa pour votre mariage ou vos soirées ...

Les cours particuliers sont la solution !

  • Vous devez prendre rendez-vous à l'avance. Les cours ont lieu dans la journée.
  • Les dates et les horaires sont à déterminer avec le professeur.
  • Vous choisissez la ou les danses que vous souhaitez apprendre.
